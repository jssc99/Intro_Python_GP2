Sur ce sujet nous avons reussi a convertir la plus part du projet en python afin de pouvoir utiliser nos class c++ a python à l'aide de Swig 
, malheuresement nous avons du mal à donnée nos instance de class c++ à python afin de pouvir les utiliser.Cependant nous pouvons toujours les appellées nos class C++ convertie en python dans python , concraitement c'est passer des donné a python depuis le c++ en runtime qui nous est problématique.
