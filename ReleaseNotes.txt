- Le Binding : 
Sur ce sujet nous avons reussi à convertir la plus part du projet en python afin de pouvoir utiliser nos classes C++ dans python à l'aide de Swig, malheuresement nous avons eu du mal à donnée nos instance de classes C++ à python afin de pouvir les utiliser. Cependant nous pouvons toujours appellée nos classes C++ convertie en python dans python, cepandant c'est passer des donnée python depuis le C++ en runtime qui nous est problématique.
Pour autant nous pouvons charger des scripts et les reload avec la touche F5 en runtime exemple avec des prints.

Il ce peux que au telechargement du projet le FichierCorePython.pyd ne se charge pas correctement pour ca il faut recompiler la librairie dans le projet PythonScript et remplacer l'ancient fichier avec le nouveau qui ce situe dans le dossier script dans le project folder.

Ps : Tout la partie python ne fonction que en Release, nous utilisons Python 3.10 / 3.11.
